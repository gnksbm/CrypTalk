# CrypTalk

### 프로젝트 소개
> 암호화폐에 관심있는 사람들을 위한 커뮤니티 앱

### 핵심기능
- 회원 인증(회원가입/로그인/토큰 갱신)
- 게시글 및 댓글 등록/수정/삭제
- 카드사 결제 및 결제내역 조회
