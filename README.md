# CrypTalk

### 프로젝트 소개
> 암호화폐에 관심있는 사람들을 위한 커뮤니티 앱
### 지원 버전
> iOS 15.0(SPM iOS 14.0)

### 핵심기능

- 회원 인증(회원가입/로그인)
- 게시글 및 댓글 등록/수정/삭제
- 시가총액 기준 코인목록 조회 / 검색
- 암호화폐 카드사 결제 및 구매내역 조회
- 다른 유저의 포트폴리오 조회
